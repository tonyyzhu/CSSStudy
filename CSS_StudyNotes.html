XHTML and CSS

There are two types of elements: unreplaceable elements such as 'div', 'a', 'p'; and replaceable emelents such as 'img'; in HTML.

The format is '<img src="picture.jpg" alt="" />'. 'src' is the attribute, 'picture.jpg' is the value.

CSS terminalogy:

Look at a typical CSS statement: h1 {color: #333; font-size: x-large; }. 'color' is called 'Property'; after ":" is the value such as '#333'. Something such as 'x-large' is also called 'key words'. The pair of "color: #333;" is called 'Declaration', which has to be terminated by ';'. The whole {color: #333; font-size: x-large; } is called 'Declaration block'. 'h1' is called 'Selector'. The whole thing (a selector plus a declaration block) is called a 'Rule'.

CSS Linking

Useing the following for embedded CSS style: <style type="text/css"> 'copy CSS file content here' </style >

Using the following for linking to external CSS file: <link rel="stylesheet" type="text/css" href="style/base.css" media="all" />
